# Companies

Ce dossier décrit le contexte professionnel de chaque employeur.

Chaque fichier présente notamment :

- l'entreprise ;
- le contexte ;
- les différents postes occupés ;
- l'évolution de carrière ;
- les principaux domaines d'intervention.

Les réalisations détaillées sont documentées dans le dossier `projects`.

Les entreprises apportent le contexte chronologique de la carrière.
