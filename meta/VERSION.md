---

## Statut

🟢 Stable

---

## Description

Première version stable de Career OS.

Cette version établit :

- la structure du dépôt ;
- les règles de fonctionnement ;
- la base de connaissances ;
- le processus de génération des livrables.

À partir de cette version, l'objectif est principalement d'enrichir la base de connaissances plutôt que de modifier l'architecture du projet.

---

## Compatibilité

Cette base de connaissances est conçue pour être utilisée avec un assistant IA capable de :

- lire des fichiers Markdown ;
- raisonner à partir d'une base de connaissances ;
- générer des documents professionnels.

Le projet reste indépendant d'un modèle ou d'un fournisseur d'IA particulier.
