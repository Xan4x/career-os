# Domaines d'expertise

Ce dossier regroupe les domaines d'expertise déduits de l'ensemble de la base de connaissances.

Contrairement aux projets, un domaine ne décrit pas une réalisation particulière.

Il représente une synthèse des connaissances, concepts, compétences et expériences acquises au travers de plusieurs projets.

Les domaines permettent notamment de :

- retrouver rapidement une expertise ;
- mettre en évidence des connaissances d'architecture ;
- adapter plus facilement un CV à une offre d'emploi ;
- éviter de dupliquer les mêmes informations dans plusieurs projets.

Les domaines sont alimentés à partir des projets.

Ils ne remplacent jamais les projets.

Lorsqu'un projet est ajouté ou enrichi, il peut être nécessaire de mettre à jour un ou plusieurs domaines d'expertise.
