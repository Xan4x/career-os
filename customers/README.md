# Clients

Ce dossier décrit les principaux clients auprès desquels j'ai réalisé des missions.

Il complète le dossier `companies`.

Une entreprise représente mon employeur.

Un client représente l'environnement dans lequel les projets ont été réalisés.

Cette distinction est particulièrement importante dans une carrière de consultant.

Chaque client décrit notamment :

- le contexte ;
- l'environnement technique ;
- les contraintes ;
- les principaux domaines fonctionnels ;
- les projets réalisés.

Les projets restent documentés dans le dossier `projects`.
