# Notes

Ce dossier contient les informations qui ne sont pas encore suffisamment structurées pour être intégrées à la base de connaissances.

Il peut s'agir :

- d'idées ;
- de souvenirs ;
- de technologies à documenter ;
- de projets à approfondir ;
- de questions en attente.

Les notes sont temporaires.

Lorsqu'elles sont suffisamment complètes, elles doivent être intégrées dans les dossiers appropriés (`projects`, `domains`, `person`, etc.).
