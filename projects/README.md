# Projects

Les projets constituent le cœur de Career OS.

Chaque projet décrit une réalisation concrète.

Un projet documente notamment :

- son contexte ;
- son objectif ;
- mon rôle ;
- les choix d'architecture ;
- les technologies utilisées ;
- les difficultés rencontrées ;
- les réalisations ;
- les résultats obtenus ;
- les compétences démontrées.

Les projets sont la principale source utilisée pour générer les livrables.

Lorsqu'un projet est enrichi, il peut être nécessaire de mettre à jour un ou plusieurs domaines d'expertise.
